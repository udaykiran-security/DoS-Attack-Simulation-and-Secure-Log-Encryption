# DoS Attack Simulation and Secure Log Encryption

## Overview

This project demonstrates the execution and analysis of Denial of Service (DoS) attack simulations in a controlled cybersecurity lab environment. The assessment includes HTTP-based Slowloris attacks, ICMP flood attacks, network traffic verification using Wireshark, system resource monitoring, and secure encryption of attack logs using OpenSSL.

The objective was to evaluate the resilience of a target system under network stress conditions and demonstrate secure handling of attack-related data.

## Objectives

* Perform network reconnaissance
* Identify target services and exposed ports
* Simulate HTTP-based DoS attacks
* Simulate ICMP flood attacks
* Capture and analyze attack traffic
* Monitor system resource utilization
* Secure attack logs using encryption
* Validate encryption and decryption processes

### Secure Log Encryption

* AES-256-CBC Symmetric Encryption
